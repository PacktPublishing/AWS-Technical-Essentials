# Chapter 9 – Capstone Project: The Vegan Studio

## Overview
In this capstone project, you'll design, architect, and build a complete three-tier web application on AWS using the services and concepts covered throughout this study guide. The project combines networking, compute, storage, databases, security, scalability, and high availability into a real-world solution.

## Project Phases
- Gather business and technical requirements.
- Design the High-Level Architecture (HLD).
- Produce the Low-Level Design (LLD).
- Validate the architecture with high availability and failover planning.
- Build the Amazon VPC and networking components.
- Create and configure the Amazon S3 bucket.
- Configure IAM roles and permissions.
- Deploy Amazon RDS with Multi-AZ for high availability.
- Configure an Elastic Load Balancer (ELB).
- Implement an Auto Scaling Group (ASG).
- Deploy and validate the three-tier web application.
- Perform high availability and failover testing.

## Learning Outcomes
By the end of this chapter, you will be able to:
- Design a production-ready three-tier architecture on AWS.
- Deploy a secure, scalable, and highly available application.
- Apply AWS architectural best practices across multiple services.
- Validate application resilience through high availability and failover testing.
- Demonstrate end-to-end AWS architecture skills in a real-world capstone project.
