# Chapter 3 – Amazon Simple Storage Service (Amazon S3)

## Overview
This chapter introduces Amazon S3, AWS's highly scalable object storage service. You'll learn the different storage types available in AWS, how Amazon S3 works, and how to configure and manage S3 buckets for common real-world use cases.

## In this chapter
- Understand the differences between block, file, and object storage.
- Learn the core concepts and features of Amazon S3.
- Explore common Amazon S3 use cases.
- Create and configure an S3 bucket.
- Enable and manage S3 Versioning.
- Configure Cross-Region Replication (CRR) for disaster recovery.
- Host a static website using Amazon S3.

## Learning Outcomes
By the end of this chapter, you will be able to:
- Explain the differences between block, file, and object storage.
- Describe the key features and benefits of Amazon S3.
- Create and securely configure S3 buckets.
- Protect data using versioning and cross-region replication.
- Deploy and host a static website using Amazon S3.
