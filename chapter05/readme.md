# Chapter 5 – Compute Services on AWS (Amazon EC2)

## Overview
This chapter introduces Amazon EC2, AWS's flagship compute service, and Amazon EBS, the block storage service for EC2 instances. You'll learn how to deploy, manage, and automate virtual servers in the AWS Cloud.

## In this chapter
- Understand the fundamentals of Amazon EC2.
- Learn about EC2 instance types, purchasing options, and use cases.
- Explore Amazon EBS and persistent block storage.
- Create and manage Amazon Machine Images (AMIs).
- Automate EC2 deployments using user data scripts.
- Review EC2 security and operational best practices.

## Learning Outcomes
By the end of this chapter, you will be able to:
- Launch and manage Amazon EC2 instances.
- Attach, manage, and resize Amazon EBS volumes.
- Create reusable Amazon Machine Images (AMIs).
- Automate instance provisioning using EC2 user data scripts.
- Select the appropriate EC2 and EBS options for different workloads.

## Labs
- Launch an Amazon EC2 instance.
- Create, attach, and manage Amazon EBS volumes.
- Create and launch instances from Amazon Machine Images (AMIs).
- Deploy EC2 instances using user data scripts to automate software installation and configuration.
