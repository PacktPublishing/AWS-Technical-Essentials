# Chapter 4 – Amazon VPC and Networking

## Overview
This chapter introduces Amazon Virtual Private Cloud (Amazon VPC) and the core networking services used to build secure, highly available AWS environments. All hands-on exercises are completed using the AWS Management Console with detailed step-by-step instructions.

## In this chapter
- Understand the fundamentals of AWS networking.
- Create and configure an Amazon VPC.
- Design public and private subnets.
- Configure route tables and internet connectivity.
- Deploy and attach Internet Gateways and NAT Gateways.
- Configure Security Groups and Network ACLs.
- Understand IP addressing and CIDR blocks.
- Explore common VPC networking architectures and best practices.

## Learning Outcomes
By the end of this chapter, you will be able to:
- Design and build a secure Amazon VPC.
- Configure networking components to enable secure communication.
- Control access using Security Groups and Network ACLs.
- Create public and private network architectures following AWS best practices.

## Labs
- All labs are performed using the **AWS Management Console**.
- Step-by-step instructions are provided for every exercise.
- No source code is required for this chapter.
