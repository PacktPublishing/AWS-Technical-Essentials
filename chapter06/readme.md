# Chapter 6 – AWS Lambda, High Availability and Scalability

## Overview
This chapter introduces AWS Lambda and the AWS services used to build highly available and scalable applications. You'll learn how Elastic Load Balancing and Auto Scaling work together to automatically distribute traffic and respond to changing demand.

## In this chapter
- Understand the fundamentals of AWS Lambda.
- Learn the principles of high availability and scalability.
- Explore Elastic Load Balancing (ELB) and its different load balancer types.
- Configure Auto Scaling Groups (ASGs).
- Understand Auto Scaling policies and scaling strategies.
- Automate infrastructure management using AWS Lambda.

## Learning Outcomes
By the end of this chapter, you will be able to:
- Explain the benefits of serverless computing with AWS Lambda.
- Design highly available architectures using Elastic Load Balancing.
- Configure Auto Scaling Groups to respond to demand automatically.
- Select the appropriate load balancer and scaling policy for different workloads.
- Automate EC2 operations using AWS Lambda.

## Labs
- Create and configure an Elastic Load Balancer (ELB).
- Configure an Auto Scaling Group (ASG).
- Create an AWS Lambda function to automatically start and stop Amazon EC2 instances.
