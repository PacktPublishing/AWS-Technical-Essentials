# Chapter 7 – Databases on AWS (Amazon RDS)

## Overview
This chapter introduces AWS database services with a primary focus on Amazon Relational Database Service (Amazon RDS). You'll learn how to deploy, manage, and secure relational databases in the AWS Cloud using the AWS Management Console.

## In this chapter
- Understand the different database types available on AWS.
- Learn the fundamentals of Amazon RDS.
- Explore supported database engines and common use cases.
- Create and configure Amazon RDS database instances.
- Understand backups, Multi-AZ deployments, and read replicas.
- Learn database security and monitoring best practices.

## Learning Outcomes
By the end of this chapter, you will be able to:
- Explain the different database options available on AWS.
- Deploy and manage Amazon RDS instances.
- Configure backups and high availability for relational databases.
- Apply security and operational best practices for Amazon RDS.

## Labs
- All labs are performed using the **AWS Management Console**.
- Step-by-step instructions are provided for every exercise.
- No source code is required for this chapter.
