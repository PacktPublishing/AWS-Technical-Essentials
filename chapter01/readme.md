# Chapter 1 – Introduction to Cloud Computing

## Overview
This chapter introduces the fundamentals of cloud computing and AWS.

## In this chapter
- Learn the core concepts of cloud computing.
- Explore the six key advantages of cloud computing.
- Understand the AWS Global Infrastructure, including Regions and Availability Zones.
- Review AWS Support Plans and when to use each one.
- Create your AWS Free Tier account.
- Apply for up to **$200 in AWS Free Tier credits** (where eligible).

## Learning Outcomes
By the end of this chapter, you will be able to:
- Explain the benefits of cloud computing.
- Describe the AWS Global Infrastructure.
- Understand the available AWS support options.
- Set up an AWS account ready for the hands-on labs throughout the course.
