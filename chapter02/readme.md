# Chapter 2 – AWS Identity and Access Management (IAM)

## Overview
This chapter introduces AWS Identity and Access Management (IAM) and the core AWS identity services used to securely manage access to AWS resources.

## In this chapter
- Understand AWS identity and access management concepts.
- Compare AWS IAM, AWS IAM Identity Center, and Amazon Cognito.
- Learn best practices for securing the AWS Root User.
- Create and manage IAM users, groups, and roles.
- Understand how IAM policies control permissions.
- Configure and access your AWS account using the AWS CLI.
- Explore additional AWS security tools and best practices.

## Learning Outcomes
By the end of this chapter, you will be able to:
- Explain the purpose of IAM, IAM Identity Center, and Amazon Cognito.
- Secure your AWS account using identity best practices.
- Create and manage users, groups, roles, and policies.
- Authenticate to AWS using the AWS CLI.
- Apply security best practices to protect your AWS environment.
